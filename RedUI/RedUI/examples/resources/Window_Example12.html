Window Display Options Example

This example explains how to use display options to control window position.

UI Model

{
    "windows": [
        {
            "name": "SimpleWindowExample",
            "title": "Simple Window Example",
            "left": 500,
            "top": 100,
            "width": 400,
            "height": 300
        }
    ]
}

Code

var exampleWindow = redui.createNewWindow("SimpleWindowExample");
exampleWindow.show({ windowPosition: redui.windowPosition.center });
