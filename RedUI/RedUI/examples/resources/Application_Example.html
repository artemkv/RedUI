RedUI Application

This page explains the anatomy of the RedUI application.

For more detailed documentation please refer to a project page on codeplex.com



<!DOCTYPE html>

<html>
<head>
    <title>RedUI Demo</title>

    <!-- Default CSS for RedUI -->
    <link href="/Content/redui-1.0.0.css" rel="stylesheet" type="text/css" />
    <!-- Default color scheme for RedUI -->
    <link href="/Content/redui-1.0.0.default.css" rel="stylesheet" type="text/css" />
</head>
<body>
    <!-- This DIV is a container for the RedUI application.
        It has to have attribute id set to "redui_application".
        Typically it should be styled to take the full size of the browser window.
        data-templatesbaseurl atribute contains url of the folder with RedUI templates.
        data-model attribute contains url of the model -->
    <div id="redui_application"
        data-templatesbaseurl="/Content/templates/"
        data-model="/examples/model.js" ></div>

    <!-- Dependencies (JQuery, JQueryUI and Mustache) excluded for brevity. -->

    <!-- RedUI framework -->
    <script src="/Scripts/redui-1.0.0.js" type="text/javascript"></script>

</body>
</html>
