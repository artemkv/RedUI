Rules:
1. Prefix with redui-

Example: redui-hidden

2. Use the control type for controls

Example: redui-combobox

3. Use the modified term for the specialized version of a control

Example: 

redui-dialogbutton for specialized redui-button
redui-modalwindow for specialilzed redui-window

4. Use postfix(es) for the dependent element

Example: redui-textbox-label, redui-checkbox-outer, redui-window-clientarea-inner

5. Use postfix to indicate the specific state of the control

Example: redui-treeviewnode-selected

6. Use special id to reference the single target element, concatenating the id of the main element with all css postfixes of the targeted element without separators

Example: {{id}}_clientareaouter for element with class redui-window-clientarea-outer
